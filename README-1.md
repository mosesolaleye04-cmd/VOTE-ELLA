# ANTICIPATE ELLA — MNE 201 CBT
### Engineering Learning Lab App | FUTA 200L

> A fully offline, mobile-ready CBT study platform for **MNE 201 (Introduction to Engineering)** at the Federal University of Technology, Akure (FUTA).

Built by **LMD Grafix** for FUTA 200L Engineering students.

## 🔗 Live App
👉 **[Open ANTICIPATE ELLA on GitHub Pages](https://YOUR-USERNAME.github.io/anticipate-ella-mne201/)**

---

## ✨ Features

- **215 questions** across 11 topic areas
- **4 study modes** — Exam, Practice, Quick Fire, Review
- **Topic filter** — drill only the topic you're weak on
- **60-second countdown timer** per question (Exam & Quick Fire modes)
- Instant answer feedback with correct answer shown
- End-of-session review of wrong & skipped questions
- Fully **offline** after first load — no internet required
- **Mobile-optimized** for iPhone & Android
- Animated **ANTICIPATE ELLA** wave letter branding

## 🚀 How to Use

1. Open the live link above (or `index.html` locally)
2. Select a **topic filter** (or keep "All Topics")
3. Choose your **study mode**
4. Hit **Start Session** and begin!
